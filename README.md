# Senora Paints

### Custom Shopify Art Store & Commission Platform

## Overview

Senora Paints is a custom Shopify website built for an abstract artist to sell original artwork, merchandise, and offer bespoke commissioned pieces.

The platform combines a traditional e-commerce experience with a **structured commission request system**, allowing customers to collaborate directly with the artist.

---

## Features

* Custom Shopify storefront
* Product catalog for artwork, mugs, and greeting cards
* Cart and checkout flow
* Artist bio and branding pages
* Contact form for general inquiries

---

## Commission System

A dedicated commissions page allows customers to submit detailed project requests, including:

* Preferred color palette
* Emotional tone and style
* Space description (where artwork will be displayed)
* Size requirements
* Budget range
* Optional premium add-ons

### Premium Add-ons

* Custom floater frames
* Metallic leaf accents
